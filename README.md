## 🎯 The Methodology

Every problem solved in this repository undergoes a "Problem Autopsy". This enforces a strict discipline:
1.  **The 15-Minute Rule:** If the core logic isn't cracked within 15 minutes, it's a fail. Time to study the optimal solution.
2.  **Complexity First:** Time and Space complexity are declared *before* writing the code. 
3.  **Identify Triggers:** Documenting the exact phrasing in the problem statement that gave away the pattern.
